# paddleocr-application
paddleocr的应用案例

用于生产日期此类数字或字母的检测。

内含多种图像形态处理，利用hash计算评分，带plc通讯案例，已经经过大量测试，只是没有时间去完善了。
有机会可以接着更新，比如利用label进行固定检测范围。
